## 🚀 Featured Projects

### 1. MyzaLab — Remote Labs Platform 
A web platform providing remote browser-based access to virtual machines, developed during my internship at Z&A IT Company. It involved full-stack development and systems automation.
* **Tech Stack:** Java EE (Servlets), Apache Guacamole, VMware ESXi, Linux (Bash), NGINX, REST APIs

### 2. Ohne —  Audio Isolation 
A powerful audio isolation tool that separates vocals from background music with an intuitive GUI and local processing.
* **Tech Stack:** Python, FFmpeg, Demucs, YT-DLP 
* **Link:** [Website](https://ohne.space/)

### 3. NBASenti — Statistical & Sentiment Analysis 
A data platform that combines NBA player statistics with real-time sentiment analysis from Twitter, using NLP models to generate insights.
* **Tech Stack:** Python, Spark, NLP (VADER, Gemini), Power BI 
* **Links:** [Demo](https://youtu.be/DzhHy9HJmE) 

### 4. Physicsly — Interactive 3D/AR E-Learning 
An interactive web application for learning physics through 3D simulations and augmented reality visualizations.
* **Tech Stack:** React, Node.js 
* **Links:** [Website](https://physicsly.com)
